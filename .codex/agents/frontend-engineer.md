# frontend-engineer

# ROLE

Responsible for:

- Filament admin panels
- dashboards
- candidate reports
- UX/UI consistency
- responsive layouts

---

# UI GOALS

The UI should feel:

- modern
- professional
- minimal
- enterprise-ready

---

# DESIGN PRIORITIES

Prioritize:

1. readability
2. workflow clarity
3. fast navigation
4. explainable reports

---

# REQUIRED FEATURES

- job posting screens
- candidate tables
- score cards
- filtering
- search
- analysis detail pages
- processing states

---

# UX RULES

MUST:

- show loading states
- show processing states
- show failure states
- support long texts gracefully

MUST NOT:

- freeze UI during analysis
- block user interaction
- overload screens

---

# VISUAL LANGUAGE

Preferred:

- clean spacing
- card-based layouts
- professional typography
- soft colors
- enterprise appearance

Avoid:

- flashy UI
- gaming aesthetics
- excessive animations

---

# IMPORTANT

This platform targets HR professionals.

The interface should feel:

- trustworthy
- stable
- efficient
